## 📌 Project Overview  

This project was developed as part of Internship Task 1 to explore and creatively remix Bootstrap 5 components into a clean, modern, and fully responsive multi-page website.

The objective was not to copy example templates, but to extract high-quality UI patterns from the official Bootstrap documentation and combine them into an original layout with strong visual composition and component synergy.

This project demonstrates practical understanding of:

- Bootstrap 5 Grid System
- Utility Classes
- Responsive Design
- Component Composition
- Layout Structuring
- UI/UX Consistency

---

## 🛠 Technologies Used  

- HTML5  
- CSS3  
- Bootstrap 5 (via CDN)  
- Bootstrap Icons  
- GitHub (Version Control)  
- Netlify (Deployment)  

No backend or database was used, as the task focuses purely on frontend UI development.

## 🧠 Development Process (Step-by-Step)

1. Explored Bootstrap 5 official documentation and examples.
2. Identified reusable UI components (navbar, cards, carousel, forms).
3. Designed wireframe layout structure.
4. Implemented Home page with hero and component sections.
5. Built Services page with grid-based composition.
6. Designed Contact page with form validation.
7. Applied custom CSS for visual enhancement.
8. Tested responsiveness across screen sizes.
9. Deployed project using Netlify.

---

## ⚡ Challenges Faced  

- Managing consistent spacing between sections.
- Ensuring proper responsive behavior of carousel.
- Implementing form validation correctly.
- Maintaining visual balance while remixing components.

These challenges were solved by carefully reviewing Bootstrap documentation and testing layouts multiple times.
