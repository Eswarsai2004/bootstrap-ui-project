## 📌 Project Overview  

This project was developed as part of Internship Task 1 to explore and creatively remix Bootstrap 5 components into a clean, modern, and fully responsive multi-page website.

The objective was not to copy example templates, but to extract high-quality UI patterns from the official Bootstrap documentation and combine them into an original layout with strong visual composition and component synergy.

This project demonstrates practical understanding of:

- Bootstrap 5 Grid System
- Utility Classes
- Responsive Design
- Component Composition
- Layout Structuring
- UI/UX Consistency

---

## 🛠 Technologies Used  

- HTML5  
- CSS3  
- Bootstrap 5 (via CDN)  
- Bootstrap Icons  
- GitHub (Version Control)  
- Netlify (Deployment)  

No backend or database was used, as the task focuses purely on frontend UI development.
